# Data Analyst Portfolio

## Portfolio Overview
This portfolio showcases data analysis projects built with Excel, PostgreSQL, Tableau, and Python. The work focuses on data cleaning, validation, KPI development, exploratory analysis, and business-facing insight generation across retail, manufacturing, and analytical chemistry use cases.

## Skills Demonstrated
- Excel analysis and KPI reporting
- SQL / PostgreSQL querying
- data cleaning and validation
- exploratory data analysis
- dashboard development
- business recommendation framing
- presentation-ready reporting
- QA and monitoring workflows

## Projects

### 1. Retail Sales Analysis
**Tools Used:** PostgreSQL, Tableau

This project analyzes retail sales performance with a focus on pricing, promotions, and revenue optimization. The workflow includes raw table creation, cleaning and type conversion, validation checks, analytical SQL development, and dashboard creation.

**What it shows:**
- SQL-based data preparation and validation
- business analysis around pricing and promotions
- dashboarding in Tableau
- translating query results into operational recommendations

### 2. LCMS Retention Analysis
**Tools Used:** Python, Jupyter Notebook

This project models LCMS retention time using molecular descriptor data. It compares multiple regression approaches and evaluates model performance using error metrics and prediction analysis.

**What it shows:**
- analytical problem solving with Python
- model comparison and evaluation
- technical interpretation of results
- communication of predictive analysis in a portfolio setting

### 3. Manufacturing Analysis
**Tools Used:** Excel

This project analyzes QC blend adjustment activity in a manufacturing environment using Excel for data consolidation, validation, KPI reporting, and blend-level monitoring. The workbook includes source consolidation, lookup logic, QA checks, helper-column development, summary pivots, and a KPI dashboard.

**What it shows:**
- Excel-based analytical workflow design
- validation and QA logic for operational data
- KPI development and trend reporting
- manufacturing-focused business analysis
